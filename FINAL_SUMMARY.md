## 🚀 Quick Start

```bash
# 1. Install
composer install

# 2. Setup
cp .env.example .env
php artisan key:generate

# 3. Configure database in .env
DB_DATABASE=student_management_advanced
DB_USERNAME=root
DB_PASSWORD=your_password

# 4. Migrate & Seed
php artisan migrate --seed

# 5. Run
php artisan serve

# 6. Visit
http://localhost:8000
```
